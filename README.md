# Excel Analyzer

This project analyzes Japanese credit card statement data.

Multiple CSV files are merged and analyzed to generate an Excel report.

## Folder Structure

excel_analyzer

src/
main.py

input/
credit card CSV files

output/
generated Excel reports

## How to Run

Place CSV files into the input folder.

Run:

python src/main.py

The Excel report will be generated in the output folder.

## Features

- merge multiple CSV files
- monthly spending summary
- category analysis
- Excel report generation
